Hover over a bar to reveal its height. Try dragging the handle in the bottom right to resize the graph.

How does it work?

The graph container is a div.bar-graph element, which has the following styling:

.bar-graph {
    display: flex;
    align-items: flex-end;
}

display: flex; displays each of the child elements as one element of a row, and align-items: flex-end; says that we want the bottom edge of each child item to line up at the “end” (i.e. the bottom) of the flex container.

The bars themselves are div.bar elements, styled like so:

.bar-graph .bar {
    flex-grow: 1;
}

flex-grow: 0; is the default. By changing this value to 1, we allow each bar’s width to grow, filling the full width of the graph.

Finally, each individual bar has inline styling dictating, as a percentage, how much of the height of the graph to consume. For example:

<div class="bar" style="height: 56%"></div>
